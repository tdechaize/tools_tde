
		Status and versions of "free" compilers C/C++ installed on Windows 11 64 bits

Prerequisite : To test and generate executable or DLL on Windows 11 64 bits with one of "free" compilers C/C++, you 
  must installed all "last" versions of different tools : 
	- all "free" (or assimiled) compilers C/C++, versions 32 bit, but also 64 bit in much case, to your choice,
	- many useful tools like generator CMAKE, SCM GIT, and some IDE like Code::Blocks (version 20.03 dated 2020 and
	  the last recent packaged evolution with "nigthly build" version 13538 june 2023 that provided MinGW64 package) 
	  or Red-Panda Cpp version 3.1 2024 with MinGW64 too,
	- and also, useful tools of GNU ported on Windows (file, ldd, make, ...) issued of Sourceforge, project "ezwinport", 
	 or much old, project "UnxUtils", or after install of CYGWIN64.

NB : InstalledDir is the full name of directory to access at all executables or tools included in "free" compilers.
	By example, if you want to use these compilers in "command line" (into console CMD or PowerShell) or with CMAKE, 
	you must add one of this "binary" directory on your variable PATH, with command :
			"set PATH=bin_directory_choosen;%PATH%".

GENERAL REMARK : With compilers that support "real" 64 bit version on Windows systems (exclude Borland C/C++, DMC and Open WATCOM), 
 if you want distinguish in your code if you construct targets to 32 bit or 64 bit, you can test variable "#ifdef _WIN64". If test
 is positive, you build targets to 64 bit version.
 In all case, if you want test if you build targets on Windows system, only test like "#ifdef _WIN32" be enougth.
 The logic is next : always define _WIN32 for all compilers (32 or 64 bit), and only if 64 bit, define _WIN64.
 With GCC/MinGW, same principle : always define __MINGW32__ for 32 or 64 bit, and only if 64 bit, define __MINGW64__.
 Only "native" GCC compiler of CYGWIN64 define __CYGWIN__ only, nothing about _WIN32 or _WIN64, and ... it's normal ! 
 Note that "native" GCC compiler can generate an "Windows" compatible target ... (apps with GUI Win32 or DLL).
 Note also that this explanation is only true on Windows systems, if you want test support of 64 bit of your targets out of Windows,
 another principes and tests must be used.

#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[])

{

#ifdef _WIN32
    printf("_WIN32 is defined with this compiler %s.\n", argv[1]);
#endif
#ifdef _WIN64
    printf("_WIN64 is defined with this compiler %s.\n", argv[1]);
#endif
#ifdef __NT__
    printf("__NT__ is defined with this compiler %s.\n", argv[1]);
#endif
#ifdef WIN32
    printf("WIN32 is defined with this compiler %s.\n", argv[1]);
#endif
#ifdef __WIN32__
    printf("__WIN32__ is defined with this compiler %s.\n", argv[1]);
#endif
#ifdef __MINGW32__
    printf("__MINGW32__ is defined with this compiler %s.\n", argv[1]);
#endif
#ifdef __CYGWIN__
    printf("__CYGWIN__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(_MSC_VER)
    printf("_MSC_VER is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__MINGW64__)
    printf("__MINGW64__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__GNUC__)
    printf("__GNUC__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__BORLANDC__)
    printf("__BORLANDC__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__TURBOC__)
    printf("__TURBOC__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__WATCOMC__)
    printf("__WATCOMC__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__LCC__)
    printf("__LCC__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__DMC__)
    printf("__DMC__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__POCC__)
    printf("__POCC__ is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__INTEL_LLVM_COMPILER)
    printf("__INTEL_LLVM_COMPILER is defined with this compiler %s.\n", argv[1]);
#endif
#if defined(__clang__)
    printf("__clang__ is defined with this compiler %s.\n", argv[1]);
#endif

    return 0;
}
